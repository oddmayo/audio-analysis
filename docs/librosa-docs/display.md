Data visualization
- specshow(data, ...) — Display spectrogram/chromagram/cqt/etc.
- waveshow(y, sr, ...) — Visualize waveform in time domain.

Axis formatting helpers: TimeFormatter, NoteFormatter, ChromaFormatter, LogHzFormatter, TonnetzFormatter, etc.
